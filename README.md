# Course Builder

A React-based interactive course building platform.

## Available Commands

In the project directory, you can run the following commands to build and run the application:

### `npm install`
Installs all required dependencies for the project. You need to run this command before starting the development server for the first time.

### `npm run dev`
Runs the app in the development mode using Vite. 
*Note: Make sure to include the space between "run" and "dev" (i.e., `npm run dev` instead of `npm rundev`).*

Open [http://localhost:5173](http://localhost:5173) to view it in your browser. The page will reload when you make changes.

### `npm run build`
Builds the app for production to the `dist` folder.
It correctly bundles React in production mode and optimizes the build for the best performance.

### `npm run preview`
Locally preview your production build. Run this command after running `npm run build` to see how your production bundle behaves before deploying.
